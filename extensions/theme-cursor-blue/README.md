# Cursor Blue Theme

A beautiful and professional dark blue theme for OctateCode IDE, inspired by Cursor IDE's signature color scheme.

## Features

- **Dark Background**: Easy on the eyes with `#00040A` base color
- **Cyan Accents**: Bright, readable cyan (`#3BC0FF`) for keywords and important elements
- **Blue Highlights**: Deep blue (`#0038E5`) for keywords and control structures
- **Perfect Contrast**: High contrast ratios for accessibility
- **Terminal Support**: Fully customized terminal colors

## Installation

This theme is built-in to OctateCode IDE. To use it:

1. Open OctateCode IDE
2. Press `Ctrl+K Ctrl+T` (or `Cmd+K Cmd+T` on Mac)
3. Search for "cursor blue"
4. Select and apply

Or set it in your settings:

```json
{
  "workbench.colorTheme": "cursor blue"
}
```

## Color Palette

### Primary Colors
- **Background**: `#00040A` (Deep Black)
- **Foreground**: `#D6E0FF` (Light Blue)
- **Accent**: `#3BC0FF` (Cyan)
- **Keyword**: `#0038E5` (Deep Blue)

### Editor Colors
- **Line Highlight**: `#00040A50` (Subtle)
- **Selection**: `#3BC0FF30` (Transparent Cyan)
- **Cursor**: `#0038E5` (Blue)

### Status Bar
- **Background**: `#203F69` (Medium Blue)
- **Foreground**: `#FFFFFF` (White)

## For Collaboration

When using OctateCode IDE collaboration features, this theme provides:
- Clear visual hierarchy for workspace files
- Distinct colors for different file types
- Easy distinction between user cursors
- Smooth visual experience during real-time editing

## Customization

You can customize this theme by modifying your VS Code settings:

```json
{
  "workbench.colorCustomizations": {
    "[cursor blue]": {
      "editor.background": "#00040A",
      "editor.foreground": "#D6E0FF"
    }
  }
}
```
